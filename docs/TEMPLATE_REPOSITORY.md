# GitHub template repository

To enable **Use this template** for adopters who want a standalone repo:

1. Open **Settings → General → Template repository** on `github.com/mitchelldawkinsjr/issue-bench`
2. Check **Template repository**

Adopters can also copy the [`template/`](template/) directory into an existing project or run:

```bash
npx issue-bench init
```

The monorepo root is for development; consumer-facing files live in `template/` and are generated by `npm run sync-template`.
